# recognizePic
这是个基于百度增值税发票识别接口的自动化发票流程处理项目
# 包含内容
包括发票信息提取，自动审批，识别交易主体，并将原始数据（发票图片）保存到mongoDB中，将交易信息保存到mysql中，
将交易关系保存在neo4j中，并会将信息统计并生成一个excel表格，统计审批情况，画图，统计审批的条目数量
# 环境
pycharm，python3.11
# 更迭历史

2.测试过小数量的发票识别，多个财务关系识别
3.测试过大数量的发票识别，batch_size = 10
